Editor Trees

The concept of our editor tree was taken from height balance and binary search concepts. This was completed with a parter as the closing project to Data Structures and Algorithms at RHIT.

Runs operations in O(logn) time using balance nodes to self balance to prevent formation of a degenerate tree which runs in O(n). Operations include search, add, and delete. Our class can run additional build and copy methods to create a string within given time restraints.

Github Link: https://github.com/kernanana/EditorTreesTeam26
